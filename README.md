# Sweet Shop Management System - Full Project (Backend + Frontend)

This archive contains a complete full-stack project for the **Sweet Shop Management System**.
Backend: Node.js + TypeScript + Express + TypeORM + SQLite
Frontend: React (TypeScript)


## How to run locally (basic)
1. Backend:
   ```
   cd backend
   npm install
   npm run dev
   ```
   API runs at http://localhost:4000

2. Frontend:
   ```
   cd frontend
   npm install
   npm start
   ```
   App runs at http://localhost:3000
